# Installation

Add Blume to a new or existing project.

```package-install
npm i blume
```

## Scaffold a project

```bash
blume init
```

This creates the minimum surface:

```txt
docs/
  index.mdx
blume.config.ts
```

:::tip
Blume works regardless of your package manager, and never requires you to set up
Astro or Tailwind yourself.
:::

## Requirements

- Node.js 20 or newer
- A `docs/` folder with at least one `.md` or `.mdx` file
