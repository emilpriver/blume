# blume

Open-source, markdown-first documentation powered by Astro and Vite.

```bash
bun add blume
blume init
blume dev
```

Drop Markdown or MDX into a `docs/` folder and Blume gives you navigation,
search, theming, and a rich component library with no app boilerplate. Build to
static HTML with `blume build`, or eject to a standalone Astro project with
`blume eject`.

See the [full documentation](https://blume.dev) and the
[repository](https://github.com/haydenbleasel/blume) for guides, the component
reference, and the CLI.
